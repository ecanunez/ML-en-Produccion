# ML Football Predictor - Release v1.0

## Resumen

La versión 1.0 corresponde al primer sistema completo de clasificación multiclase para la predicción de resultados de partidos de fútbol utilizando información histórica obtenida mediante scraping desde Transfermarkt.

El proyecto cubre todo el ciclo de Machine Learning:

* Recolección de datos.
* Construcción del dataset.
* Ingeniería de variables.
* Selección de características.
* Entrenamiento y evaluación de modelos.
* Optimización de hiperparámetros.
* Pipeline de inferencia.

---

## Dataset

### Fuente

Transfermarkt

### Tamaño

* 12.599 partidos históricos.

### Variable objetivo

Resultado del partido:

* Home Win
* Draw
* Away Win

---

## Ingeniería de Variables

Se construyeron variables relacionadas con:

* Valor de mercado de los equipos.
* Diferencia de valor entre equipos.
* Forma reciente.
* Estadísticas históricas.
* Indicadores relativos entre local y visitante.

---

## Selección de Variables

Se realizaron experimentos utilizando:

* Top 10 variables.
* Top 20 variables.
* Top 40 variables.
* Dataset completo.

La selección final se basó en maximizar F1 Macro mediante validación cruzada.

---

## Modelos Evaluados

* Logistic Regression
* Random Forest
* Gradient Boosting

---

## Mejor Modelo

Random Forest optimizado.

Parámetros:

* n_estimators = 500
* max_depth = 10
* min_samples_split = 2
* min_samples_leaf = 5
* class_weight = balanced

Resultado obtenido:

* F1 Macro CV ≈ 0.472

## Limitaciones

* El sistema utiliza únicamente información histórica disponible en el dataset.
* No incorpora alineaciones, lesiones o cuotas de apuestas.
* No realiza aún scoring automático de partidos futuros.
